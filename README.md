# Mathsonics

A web app to teach math for class 6 and above, featuring an AI assistant that answers math questions only.

## Features

- Interactive lessons on various math topics
- AI chat assistant for math queries
- Simple and clean interface

## Getting Started

1. Install dependencies: `npm install`
2. Run the development server: `npm run dev`
3. Open your browser to `http://localhost:5173`

## Usage

- Browse the math topics to learn.
- Use the AI assistant to ask math questions and get answers.

## Troubleshooting

- Ensure you have Node.js installed.
- If the AI doesn't respond, check if the question is a valid math expression.
